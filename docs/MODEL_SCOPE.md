# Model Scope

Apocalypse Clock is a scenario-based analytical model for examining global systemic risk under specified assumptions. It is intended to support transparency, inspection, criticism, and structured reasoning under deep uncertainty.

The model is not a prophecy. It is not a deterministic prediction. It is not an empirical probability of civilizational collapse. It should not be treated as legal, financial, medical, security, or policy advice.

The highlighted year should be interpreted as a model-derived dynamic-cascade horizon under specified assumptions. It is a model output conditioned on the selected inputs, structure, thresholds, uncertainty ranges, and scenario logic.

Thresholds used by the model are analytical anchors, not physical tipping points. Dataset values and uncertainty ranges are model inputs, not exact real-world measurements.

The model is designed to make assumptions inspectable and contestable, not to claim certainty about future events.

# Apocalypse Clock Model Invariants

The following properties define the scientific and interpretive identity
of the Apocalypse Clock model.

Changing them changes the model itself.

## Headline rule

The main displayed year is:
- Dynamic Cascade P90

It is NOT:
- median collapse year
- deterministic forecast
- probability of extinction
- exact prediction

## Structural invariants

The model must preserve:
- 23 threats
- 8 metrics per threat
- four aggregation systems
- Monte Carlo simulation architecture
- dependency-network propagation logic
- scenario-conditioned threat enrichment

## Numerical invariants

Never silently change:
- threshold policy
- growth-rate interpretation
- quantile definitions
- p10/p50/p90 semantics
- sentinel-year handling (>2100)
- weighted scoring logic
- cascade propagation thresholds
- domain weighting normalization

## Interpretive invariants

The model output represents:
- systemic-risk horizon estimation
- uncertainty-aware scenario analysis
- probabilistic structural dynamics

The model output does NOT represent:
- deterministic prediction
- prophecy
- empirical probability of collapse
- guaranteed timeline

## Governance rule

Any modification affecting:
- headline year
- Monte Carlo distributions
- cascade logic
- quantile logic
- threshold handling
- growth-rate interpretation

must be treated as a MODEL CHANGE,
not a cosmetic or UI change.
